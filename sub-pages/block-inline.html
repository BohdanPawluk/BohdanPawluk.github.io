Block and Inline Elements

Block Elements

This is a block element. It is contained within a division tag.
The content inside a block element typically starts on a new line and takes up the full width available.

Inline Elements

This is an inline element. It is contained within a paragraph tag.
The content inside an inline element typically does not start on a new line
and only takes up as much width as necessary.
